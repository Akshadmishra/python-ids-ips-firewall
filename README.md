# Python IDS/IPS Firewall 🔥

## 📌 Overview
A Python-based Intrusion Detection and Prevention System (IDS/IPS) built using Scapy.

## 🚀 Features
- Detects brute-force attacks on multiple ports (RDP, SSH, FTP, HTTP)
- Tracks attacker IP and connection attempts
- Automatically blocks malicious IPs using Windows Firewall
- Real-time packet monitoring
- Logs attack activity

## 🛠️ Tech Stack
- Python
- Scapy
- Networking
- Windows Firewall

## 🧪 Testing
- Simulated attacks using Kali Linux
- Verified blocking using netcat and nmap
- Observed connection timeouts after blocking
